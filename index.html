Simulating birth and death

Software for simulating birth and death processes for big particle ensembles

Download this project as a .zip file Download this project as a tar.gz file

Welcome to our project page!

The goal of the project is allowing simulations wide class of birth and death processes in a microdynamic regime. These can be seen as a system of interacting particles where possible events include creation of new particle(birth) or removal of an existing one (death). The occurrence of the events is dependent on every particle and can be either deterministic or stochastic.

Please note that jumps (i.e. moving a particle from one place to another in a discrete moment) is a combination both death and birth, and is also supported by the program.

Out of many, possible applications include biological cell dynamics (cancer evolution, pattern formation), ecology (spreading of species), models from statistical physics or mathematics.

Main goals

  • graphical and command-line user interface
  • continuous space and time (up to the numerical precision)
  • 2D and 3D space (GUI) or multidimensional space (CLI)
  • various models with different levels of generality
  • low memory footprint, overall efficiency
  • available for Windows, Linux and other compatible operating systems

Build requirements

  • cmake
  • C++ compiler with C++11 support (tested on g++ 4.8)
  • boost 1.54
  • Qt5 (GUI)
